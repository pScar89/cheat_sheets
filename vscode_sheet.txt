# VSCode Keyboard Shortcuts and Features Cheat Sheet

# General Shortcuts
Ctrl + Shift + P        # Show Command Palette
Ctrl + P                # Quick Open
Ctrl + Shift + N        # New Window/Instance
Ctrl + W                # Close Window
Ctrl + ,                # User Settings
Ctrl + K Ctrl + S       # Keyboard Shortcuts

# Editing
Ctrl + X                # Cut line (empty selection)
Ctrl + C                # Copy line (empty selection)
Ctrl + V                # Paste
Ctrl + /                # Toggle line comment
Ctrl + ]                # Indent line
Ctrl + [                # Outdent line
Alt + Up/Down           # Move line up/down
Ctrl + Shift + K/L      # Select all occurrences of current word
Ctrl + F                # Find
Ctrl + H                # Replace

# Navigation
Ctrl + G                # Go to Line...
Ctrl + T                # Show all Symbols
Ctrl + Shift + O        # Go to Symbol...
Ctrl + Shift + M        # Show Problems panel
F8                      # Go to next error or warning
Shift + F8              # Go to previous error or warning
Ctrl + Shift + E        # Show Explorer / Toggle focus
Ctrl + Shift + F        # Show Search

# File Management
Ctrl + N                # New File
Ctrl + O                # Open File...
Ctrl + S                # Save
Ctrl + Shift + S        # Save As...
Ctrl + K S              # Save All
Ctrl + Tab              # Open next
Ctrl + Shift + Tab      # Open previous
Ctrl + K P              # Copy path of active file
Ctrl + K R              # Reveal active file in Explorer
Ctrl + K O              # Show active file in new window/instance

# Display
Ctrl + =/-              # Zoom in/out
Ctrl + B                # Toggle Sidebar visibility
Ctrl + Shift + D        # Show Debug
Ctrl + Shift + X        # Show Extensions
Ctrl + Shift + G        # Show Source Control
Ctrl + Shift + V        # Open Markdown preview
F11                     # Toggle full screen
Shift + Alt + 1         # Toggle editor layout (horizontal/vertical)

# Debugging
F9                      # Toggle breakpoint
F5                      # Start/Continue
Shift + F5              # Stop
F11                     # Step into
Shift + F11             # Step out
F10                     # Step over
Ctrl + K I              # Show hover

# Integrated Terminal
Ctrl + `                # Show integrated terminal
Ctrl + Shift + `        # Create new terminal
Ctrl + C                # Copy selection
Ctrl + V                # Paste into active terminal
Ctrl + Up/Down          # Scroll up/down
Shift + PgUp/PgDn       # Scroll page up/down
Ctrl + Home/End         # Scroll to top/bottom

# Features
- IntelliSense: Autocompletion, error detection, and quick fixes.
- Git Integration: Direct Git operations within VSCode.
- Extension Marketplace: Access to a vast library of extensions.
- Debugging: Built-in debugger for various languages.
- Customization: Highly customizable, from themes to settings.
- Integrated Terminal: Built-in terminal for command-line access.
- Snippets: Reusable code templates.
- Split Views: Multiple editors open at once.

# Note: Shortcuts may vary based on keyboard layout, OS, and VSCode version.
